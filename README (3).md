# Task 2 - Phishing Email Analysis
## Elevate Labs Cybersecurity Internship

---

## Objective
Identify phishing characteristics in a suspicious email sample and document the findings.

---

## Tools Used
- MXToolbox Header Analyzer (mxtoolbox.com)
- GitHub (phishing sample source)
- Manual inspection

---

## Phishing Indicators Found
1. Spoofed sender domain
2. Urgency/threatening language
3. Generic greeting
4. Mismatched URL
5. Raw IP address redirect
6. Brand impersonation
